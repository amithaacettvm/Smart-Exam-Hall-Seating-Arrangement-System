# Exam Hall Seating Arrangement Automation System

## Project Description

This project automates the process of allocating seats to students during examinations.

Users can enter student registration numbers and branches, and the system generates a randomized seating arrangement automatically.

## Features

- Add student registration numbers
- Select student branch
- Store student records
- Generate seating arrangement automatically
- User-friendly interface
- Responsive design using Bootstrap

## Technologies Used

- HTML
- CSS
- JavaScript
- Bootstrap 5

## How to Run

1. Download the project files
2. Open index.html in any web browser
3. Add student details
4. Click "Generate Seating Arrangement"
